# Cache
This is basically a rewrite of [fastcache](https://github.com/VictoriaMetrics/fastcache) with a few variations in design, with the intention of being self-educational.

If you're looking for a production-ready cache library, consider using [fastcache](https://github.com/VictoriaMetrics/fastcache).
